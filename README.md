# MacOS - FindEmails

This is a group of scripts which was used to retrieve emails from the Mail application on MacOS. You must execute the script on the MacOS where you want to retrieve the emails. 

Stack : shell, python

## List and description of the scripts

**script1-findMail.sh**

> Find all emails in the MacOS and store all email's path in a text file "mailPathList.txt" in the current directory.
